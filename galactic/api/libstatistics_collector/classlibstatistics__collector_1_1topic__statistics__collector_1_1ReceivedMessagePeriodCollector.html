Detailed Description

template<typename T>
class libstatistics_collector::topic_statistics_collector::ReceivedMessagePeriodCollector< T >

Class used to measure the received messsage, tparam T, period from a ROS2 subscriber. This class is thread safe and acquires a mutex when the member OnMessageReceived is executed.

Template Parameters
Tthe message type to receive from the subscriber / listener
